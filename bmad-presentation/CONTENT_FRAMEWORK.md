## Quality Assurance Framework (slide 19 deep content)

| Layer | Mechanism | Outsourcing Value |
|---|---|---|
| **1. Validation Gates** | Implementation Readiness Check, PM/Architect/PO/Story DoD checklists | No code without solid plan |
| **2. Adversarial QA** | Forced bug-finding review + Edge Case Hunter + Cross-Model Review | AI checks AI — humans review evidence |
| **3. TDD** | Red-green-refactor enforced, Test Architect module, Brownfield regression | No merge without passing tests |
| **4. Traceability** | Git-versioned artifacts, Control Manifests, Merge-Readiness Packs | Client sees full audit trail: requirement → code |
